# 更新履歴

## 1.0.6 (2014/10/2??)

* 変化手順に対応

	* Kifu for Flash等とほぼ同じ仕様
* 反則勝ち/負けへの対応

* 駒落ちや詰将棋などの，初期局面指定に対応
